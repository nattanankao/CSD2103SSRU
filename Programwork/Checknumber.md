# README - Checknumber

## ชื่อโปรแกรม
Checknumber (โปรแกรมตรวจสอบเลขคู่หรือเลขคี่)

## Input
ผู้ใช้ป้อนข้อมูลจำนวนเต็ม 1 ค่า
- ตัวเลข (Number)

## Process
1. รับค่าตัวเลขจากผู้ใช้
2. ตรวจสอบว่าตัวเลขหารด้วย 2 ลงตัวหรือไม่
   ```
   number % 2 == 0
   ```
3. ถ้าหารด้วย 2 ลงตัว แสดงว่าเป็นเลขคู่ (Even number)
4. ถ้าไม่หารด้วย 2 ลงตัว แสดงว่าเป็นเลขคี่ (Odd number)

## Output
แสดงผลว่า
- Even number (เลขคู่)
- Odd number (เลขคี่)

## วิธีรันโปรแกรม

1. เปิดโปรแกรม Visual Studio Code หรือ IntelliJ IDEA
2. เปิดไฟล์ `Checknumber.java`
3. Compile โปรแกรม
   ```
   javac Checknumber.java
   ```
4. Run โปรแกรม
   ```
   java Checknumber
   ```
5. ป้อนตัวเลขที่ต้องการตรวจสอบ

### ตัวอย่างการทำงาน

ตัวอย่างที่ 1

```
Enter number: 8
Even number
```

ตัวอย่างที่ 2

```
Enter number: 15
Odd number
```

## สิ่งที่ได้เรียนรู้
- การรับข้อมูลจากผู้ใช้ด้วย `Scanner`
- การใช้ตัวแปรชนิด `int`
- การใช้ตัวดำเนินการหารเอาเศษ (`%`) เพื่อตรวจสอบเลขคู่และเลขคี่
- การใช้คำสั่ง `if-else` เพื่อตรวจสอบเงื่อนไข
- การแสดงผลด้วย `System.out.println()`
- การปิด `Scanner` ด้วย `sc.close()`
