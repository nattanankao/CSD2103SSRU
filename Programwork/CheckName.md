# README - CheckName

## ชื่อโปรแกรม
CheckName (โปรแกรมค้นหาชื่อในอาร์เรย์)

## Input
ผู้ใช้ป้อนข้อมูล 1 ค่า
- ชื่อที่ต้องการค้นหา (Name)

## Process
1. สร้างอาร์เรย์เก็บรายชื่อ
   ```
   Somchai
   Somsri
   Anan
   Malee
   Nida
   ```
2. รับชื่อที่ต้องการค้นหาจากผู้ใช้
3. ใช้คำสั่ง `for` วนลูปตรวจสอบชื่อในอาร์เรย์
4. เปรียบเทียบชื่อด้วย `equalsIgnoreCase()` เพื่อให้ค้นหาได้โดยไม่สนใจตัวพิมพ์เล็ก-ใหญ่
5. หากพบชื่อ กำหนดค่า `found = true`
6. แสดงผลการค้นหา
   - พบชื่อ แสดง `Found`
   - ไม่พบชื่อ แสดง `Not Found`

## Output
แสดงผลการค้นหา
- Found
- Not Found

## วิธีรันโปรแกรม

1. เปิดโปรแกรม Visual Studio Code หรือ IntelliJ IDEA
2. เปิดไฟล์ `CheckName.java`
3. Compile โปรแกรม
   ```
   javac CheckName.java
   ```
4. Run โปรแกรม
   ```
   java CheckName
   ```
5. ป้อนชื่อที่ต้องการค้นหา

### ตัวอย่างการทำงาน

ตัวอย่างที่ 1

```
Enter name to search: Anan
Found
```

ตัวอย่างที่ 2

```
Enter name to search: John
Not Found
```

ตัวอย่างที่ 3

```
Enter name to search: somchai
Found
```

> โปรแกรมสามารถค้นหาได้แม้ว่าผู้ใช้จะพิมพ์ตัวพิมพ์เล็กหรือตัวพิมพ์ใหญ่ เนื่องจากใช้ `equalsIgnoreCase()` ในการเปรียบเทียบ

## สิ่งที่ได้เรียนรู้
- การรับข้อมูลจากผู้ใช้ด้วย `Scanner`
- การสร้างและใช้งานอาร์เรย์ (`Array`)
- การใช้ลูป `for` เพื่อค้นหาข้อมูลในอาร์เรย์
- การเปรียบเทียบข้อความด้วย `equalsIgnoreCase()`
- การใช้ตัวแปร `boolean` เพื่อเก็บสถานะการค้นหา
- การใช้คำสั่ง `if-else` เพื่อแสดงผลตามเงื่อนไข
- การปิด `Scanner` ด้วย `sc.close()`
