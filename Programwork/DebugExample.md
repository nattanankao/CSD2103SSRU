# README - DebugExample

## ชื่อโปรแกรม
DebugExample (โปรแกรมแสดงข้อมูลในอาร์เรย์)

## Input
โปรแกรมนี้ไม่มีการรับข้อมูลจากผู้ใช้ โดยกำหนดข้อมูลไว้ในอาร์เรย์ดังนี้

```
{10, 20, 30, 40, 50}
```

## Process
1. สร้างอาร์เรย์ `numbers` เพื่อเก็บตัวเลขจำนวนเต็ม 5 ค่า
2. ใช้ลูป `for` วนตั้งแต่ตำแหน่งแรก (`index = 0`) จนถึงตำแหน่งสุดท้ายของอาร์เรย์ (`numbers.length - 1`)
3. แสดงค่าของสมาชิกแต่ละตัวในอาร์เรย์ด้วย `System.out.println()`

## Output
แสดงตัวเลขทั้งหมดในอาร์เรย์ทีละบรรทัด

```
10
20
30
40
50
```

## วิธีรันโปรแกรม

1. เปิดโปรแกรม Visual Studio Code หรือ IntelliJ IDEA
2. เปิดไฟล์ `DebugExample.java`
3. Compile โปรแกรม
   ```
   javac DebugExample.java
   ```
4. Run โปรแกรม
   ```
   java DebugExample
   ```

### ตัวอย่างการทำงาน


## สิ่งที่ได้เรียนรู้
- การสร้างและใช้งานอาร์เรย์ (`Array`)
- การใช้ลูป `for` เพื่อเข้าถึงสมาชิกในอาร์เรย์
- การใช้ `numbers.length` เพื่อกำหนดจำนวนรอบของการวนลูป
- การแสดงผลข้อมูลด้วย `System.out.println()`
- การเขียนโปรแกรมที่สามารถนำไปใช้ฝึกการ Debug และตรวจสอบค่าภายในอาร์เรย์
